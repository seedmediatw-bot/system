# 種子創意 · 內部系統

公司內部 HR 與 CRM 雙核心系統。資料即時同步至 Notion。

---

## 主要功能

### HR 模組

- **請假申請** — 填寫假別、日期、天數、原因，同步 Notion
- **報帳申請** — 多筆費用明細、AI 發票辨識（支援台灣統一發票 QR Code）、收據拍照
- **員工入職** — 填寫基本資料與緊急聯絡人，自動建立員工頁面
- **員工簽到** — 上下班時間記錄
- **儀表板** — 本月請假數、待審報帳、在職員工數、本週簽到數
- **審核追蹤** — 主管核准 / 駁回，即時更新狀態

### CRM 模組

- **客戶總覽** — 狀態、優先級、跟進日期一覽，點擊開啟詳情彈窗
- **新增 / 更新客戶** — 上傳名片或會議記錄，AI 自動擷取欄位
- **批次 AI 匯入** — 貼上文字或上傳截圖，一次識別多位客戶
- **拜訪記錄** — 在客戶總覽彈窗內新增拜訪摘要、承諾事項、下一步行動，自動回填主檔
- **AI 助理** — Gemini 對話，協助操作與查詢
