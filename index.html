Notion 設定

設定 Notion API 串接，讓申請資料自動同步到 Notion

Integration Token

取得 Notion Token

  1. 前往 notion.so/my-integrations
  2. 點擊你建立的 Integration → Manage
  3. 在 Configuration 頁面點「Show」複製 Token（開頭為 ntn_）
  4. 到 Content access 頁面點「Edit access」→ 搜尋「HR 自動化系統」→ 儲存
