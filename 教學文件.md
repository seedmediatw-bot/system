**你說：**

## 三、本機測試環境設定

### 資料夾結構
```
HR/
├── index.html              ← 主要網頁（上傳 GitHub 用）
├── HR-APP.html             ← 本機備份
├── server.js               ← 本機代理伺服器
├── cloudflare-worker.js    ← Cloudflare 部署用
├── .env                    ← API 金鑰（不可上傳！）
├── .gitignore              ← 排除敏感檔案
├── .github/workflows/      ← GitHub Pages 自動部署
├── package.json
└── node_modules/
```

### 安裝套件
```bash
npm install
```

### 設定 .env
```
NOTION_TOKEN=ntn_你的token
CLAUDE_API_KEY=sk-ant-你的key（選填）
PORT=3000
```

### 啟動本機伺服器
```bash
node server.js
```
或雙擊 `start.bat`

### 測試連線
開啟瀏覽器前往：`http://localhost:3000/api/status`
應該回傳：`{"ok":true,"notion":true,...}`

## 四、部署 Cloudflare Worker

1. 前往 [workers.cloudflare.com](https://workers.cloudflare.com) 登入
2. 點「Workers & Pages」→「Create」→「Start with Hello World!」
3. 取個名字，點「Deploy」
4. 部署完成後點「Edit Code」
5. 把編輯器內所有程式碼**全部刪除**
6. 貼上 `cloudflare-worker.js` 的完整內容
7. 點「Save and Deploy」

### 設定環境變數（金鑰）
1. 進入 Worker 頁面 → 點「Settings」
2. 找到「Variables and Secrets」→ 點「Add」
3. 新增：
   - `NOTION_TOKEN` = `ntn_你的token`
   - `CLAUDE_API_KEY` = `sk-ant-你的key`（選填）
4. 點「Deploy」儲存

### 確認 Worker 正常
開啟：`https://你的worker名稱.workers.dev/api/status`
應該回傳：`{"ok":true,"notion":true,...}`

## 常見問題

**Q：按鈕沒有反應？**  
A：滾動頁面讓按鈕離開右下角 AI 助理按鈕的遮蓋範圍。

**Q：連線失敗：無法接觸伺服器？**  
A：用 `file://` 直接開啟 HTML 無法使用，請改用 `http://localhost:3000`（本機）或 GitHub Pages 網址。

**Q：Notion 顯示 false？**  
A：Cloudflare Worker 環境變數未設定，前往 Worker Settings → Variables 確認 `NOTION_TOKEN` 已填入。

**Q：更改密碼？**  
A：在 `index.html` 搜尋現有密碼數字，替換成新密碼，再 `git push`。

**Q：審核追蹤沒有資料？**  
A：點「↻ 重新整理」，或確認 Notion 資料庫已有資料且 Worker 連線正常。

**Q：更新 Cloudflare Worker 程式碼？**  
A：回到 Cloudflare → Workers & Pages → 選你的 Worker → Edit Code → 貼上新程式碼 → Save and Deploy。
